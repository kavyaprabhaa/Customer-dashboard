# Customer Business Dashboard
## Project Objective
To design and develop a Business Dashboard that visualizes customer satisfaction levels and detailed customer insights. To present data effectively using interactive charts, bar graphs, pie charts, and analytics tools for better decision-making.
To enable businesses to monitor performance, understand trends, and improve customer experience strategically.
